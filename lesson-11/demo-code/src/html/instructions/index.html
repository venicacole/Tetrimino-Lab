Lab Instructions

In lesson 1.11, you worked to debug several issues with the Tetrimino website, including correcting file paths, fixing favicon SVG icons, and connecting on-page renders of the same SVG. Your lab assignment is as follows:

Task #1

Follow the patterns to correct the Favicon image used on the "View Pieces" page. Remember that SVGs used as favicons and SVGs used in regular page content are incorporated using different elements.

Task #2

Using Flexbox, create a more visual-friendly styling for the aside content. You are not to edit the aside HTML. To style the color-preview class, you will need to practice using pseudo-classes.

Task #3

We added a custom font to each page of our website. However, I would like the custom font applied to only the text in the header, the page titles (h1), and the heading of the aside. Think critically about how best to "revert" the custom font styling. I recommend referencing the CSS syntax for font-family as a breadcrumb.

Task #4

Include a lab.md file at the root of your project and write a short paragraph that answers the following questions: What wins or better understanding did you have at the end of this lab? Which concepts of the lab do you still feel you need to work on?

(Optional) Deeper Learning Debug:

You will notice some irregular spacing on this page within the paragraph elements. What is causing that spacing and what would be a simple fix? Use your developer tools, your debug process, and pattern matching/non-matching to think critically about a solve.
